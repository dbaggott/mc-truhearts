Toggle the HP readout on/off in-game.

- New keybind **Toggle HP overlay** under **Controls → TruHearts**. Unbound by default — assign it to whatever key you like.
- Preference persists across game sessions in `config/truhearts.json`.
- Action-bar message ("TruHearts: on/off") echoes the state change so you know it took.

No changes to how the readout looks or where it sits when it's on.
